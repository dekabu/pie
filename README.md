# PiE
## About
High-level interpreted programming language for mathematical calculations
## Install
1. get the source and go to the directory
   ```
   git clone https://github.com/dekabu/pie
   cd pie
   ```
2. configure
   ```
   ./configure.sh
   ```
3. build
   ```
   make
   ```
5. test (optional)
   ```
   make test
   ```
6. install
   ```
   make install
   ```
## Uninstall
```
make uninstall
```
